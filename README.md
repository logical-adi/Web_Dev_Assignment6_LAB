# 📱 Contact Management System

## 📌 Project Overview

This is a simple and beginner-friendly **Contact Management System** built using **HTML, CSS, and JavaScript**.
It allows users to manage their contacts efficiently with basic features like adding, editing, renaming, and deleting contacts.

This project is ideal for beginners who want to understand how JavaScript interacts with the DOM.

---

## 🚀 Features

* ➕ Add new contacts
* ✏️ Edit existing contacts
* 🔄 Rename contacts
* ❌ Delete contacts
* 🎨 Simple and clean user interface
* ⚡ Real-time updates without page reload

---

## 🛠️ Technologies Used

* **HTML** – Structure of the webpage
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic

---

## 📂 Project Structure

```
Contact-Manager/
│── index.html   # Main file (HTML + CSS + JS)
```

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. The project will open in your browser

---

## 🧠 How It Works

* Contacts are stored in a JavaScript array
* Each contact contains:

  * Name
  * Phone number
* JavaScript dynamically updates the UI using DOM manipulation
